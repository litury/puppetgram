# Фильтрация дубликатов для парсера похожих каналов

## Назначение

Эта директория содержит файлы с уже спарсенными каналами, которые будут исключены из результатов поиска похожих каналов для избежания дубликатов.

## Структура директорий

```
src/modules/channelSimilarityParser/
├── exports/                          # Результаты парсинга
│   ├── channel_usernames_*.txt       # Файлы с юзернеймами
│   └── channel_detailed_*.md         # Подробные отчеты
└── filter-duplicates/                # Фильтрация дубликатов
    ├── README.md                     # Этот файл
    └── parsed-channels/              # Файлы с уже спарсенными каналами
        ├── example-parsed.txt        # Пример файла
        ├── twitereda_parsed_*.txt    # Результаты предыдущих парсингов
        └── ...                       # Другие файлы для фильтрации
```

## Формат файлов

Файлы должны содержать юзернеймы каналов, по одному на строку, в формате:

```
@channel_name1
@channel_name2
@channel_name3
```

## Как использовать

1. **Добавьте файлы** с результатами предыдущих парсингов в папку `parsed-channels/`
2. **Запустите парсер** - он автоматически загрузит все файлы из этой папки
3. **Каналы из файлов фильтрации** будут исключены из новых результатов
4. **Новые результаты автоматически** добавляются в фильтр для будущих запусков

## Поддерживаемые форматы

- `.txt` файлы с юзернеймами (по одному на строку)
- Юзернеймы могут быть с символом `@` или без него
- Пустые строки и строки, начинающиеся с `#`, игнорируются

## Примеры файлов

### Результаты предыдущих парсингов

- `twitereda_parsed_2025-07-14.txt`
- `gamedev_channels_parsed_2025-07-13.txt`
- `qa_channels_parsed_2025-07-12.txt`

### Ручные исключения

- `blacklist.txt` - каналы, которые не должны попадать в результаты
- `spam_channels.txt` - спам каналы для исключения

## Автоматическое добавление

Парсер автоматически добавляет новые результаты в фильтр:

- Создается файл `{sourceChannel}_parsed_{date}.txt`
- Файл содержит все найденные юзернеймы
- При следующем запуске эти каналы будут исключены
